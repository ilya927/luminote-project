About This Project

This project is a Spotify-inspired music streaming website, designed to provide users with a smooth and modern listening experience. It features interactive album sections, clean audio controls, and a sleek, responsive layout styled using Bootstrap.

The project was created and developed by Ilia Chachibaia over the course of approximately one and a half months. It combines creativity, technical skill, and a deep passion for music and web development.

Technologies used: HTML, CSS, Bootstrap, JavaScript, and Flask.
